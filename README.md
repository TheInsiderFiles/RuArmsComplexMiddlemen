# RuArmsComplexMiddlemen

Российские оборонные компании для закупок санкционной продукции пользуются услугами закупочных фирм-посредников. Изучение данных об уплате НДС позволило _The Insider_ установить значительное число компаний, которые, ведя внешнеэкономическую деятельность, также совершают сделки с оборонной промышленностью. 


В данном репозитории содержится список ведущих компаний-посредников по данным 2024-2025 годов.

-----

**`MIC-suppliers.csv`**

| Поле      | Описание                                                                 |
|-------------------|-----------------------------------------------------------------------------|
| `Tax Number`        | Идентификационный номер налогоплательщика (ИНН)    |
| `Company name`             | Короткое название компании   |
